# README_HUMAN

这是给人类快速阅读和操作的入口页。

## 基础指令
- “整理文章”
- “整理新文章并分类移动”
- “检索 XX 主题的文章”
- “总结 XX 主题”
- “修复链接”
- “更新索引”
- “重新整理已处理文章”
- “我最近在关注什么？”
- “反思 / 举一反三（基于 summary 生成新概念）”

## 使用说明
- 新文章可先直接写，后续再让 AI 整理。
- 处理后的文章应移动到 `/00_RAW/<分类>/`。你也可以手动在 `/00_RAW` 下新建一层文件夹，后续 AI 会直接把它当作合法分类使用。
- 如果文章路径变化，相关超链接必须同步修复。
- 反思产出默认写入 `/02_SUMMARY`，不创建新的 RAW 原文。
